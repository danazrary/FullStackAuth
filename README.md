

# FullStackAuth

This is a full-stack authentication project built using React and Express. It includes features for user registration and login, with password hashing for secure storage.

## Features

- User registration
- User login
- Password hashing

## Tech Stack

- Frontend: React
- Backend: Express

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/danazrary/FullStackAuth.git
   cd FullStackAuth
   ```

2. Install dependencies for both frontend and backend:

   ```bash
   cd frontend
   npm install
   cd ../backend
   npm install
   ```

### Running the Application

1. Start the backend server:

   ```bash
   cd backend
   npm start
   ```

2. Start the frontend development server:

   ```bash
   cd frontend
   npm start
   ```
